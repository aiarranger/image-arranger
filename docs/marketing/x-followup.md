## JA

ローンチ投稿で入れきれなかった image-arranger の細かい便利機能を3つ。

1. Cmd+K / Ctrl+K で、コマンドと全エントリのタイトル・プロンプトを横断検索できます。
2. 重要な生成には品質ゲートを付けられます。見えているベースパーツだけを参照と比較し、違えば再試行します。
3. A1111 / NovelAI / ComfyUI の PNG メタデータが残っていれば、アップロード時にプロンプトを自動入力します。

生成するツールではなく、制作判断をなくさないための管理レイヤーです。

GitHub: https://github.com/aiarranger/image-arranger

## EN

Three small image-arranger features that did not fit in the launch post:

1. Cmd+K / Ctrl+K searches commands plus every entry title and prompt across the deck.
2. Quality gates can compare visible generated parts against adopted Base references and retry important generations.
3. PNG metadata from A1111 / NovelAI / ComfyUI can auto-fill prompt fields on upload.

It still does not generate. It keeps the reference, candidate, prompt, and queue decisions from disappearing into folders.

GitHub: https://github.com/aiarranger/image-arranger
