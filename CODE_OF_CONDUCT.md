# Code of Conduct

image-arranger welcomes constructive technical discussion and contributions.

Contributors and maintainers are expected to:

- Treat other people with respect.
- Keep feedback specific, actionable, and relevant to the project.
- Avoid harassment, personal attacks, discriminatory language, and disclosure of private information.
- Respect licensing, attribution, and usage limits for prompts, generated media, and reference assets.

Reports should go to the maintainer contact listed in `SECURITY.md` once the public repository is created.
